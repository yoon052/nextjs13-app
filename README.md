# next.js 를 이용한 간단한 포스트앱 만들기

<br>

- pocketbase, nextjs 를 통해 포스트앱을 구현했습니다.

</br>

- [x] 추가한 포스트들을 pocketbase 에서 확인할 수 있습니다.

- [x] input 폼을 통해 간단한 포스트를 추가하고, 페이지 자동 새로고침을 구현했습니다.
